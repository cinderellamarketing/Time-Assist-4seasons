How your CRM is set up

▶

Your Capsule CRM is organised around four areas that mirror how your business works:

Contacts (People & Organisations) covers everyone you interact with: customers, prospects, suppliers, networking contacts, referral partners. Each person can be linked to an organisation.

Sales Pipeline tracks every enquiry from first contact through to won or lost. This is the same process regardless of which service the enquiry is for.

Projects (Customer Work board) is where an opportunity goes once it has been won. Projects are organised by service type: Grounds Maintenance, Landscaping, Fencing, Tree Work.

Tasks are the follow-ups, reminders, and actions assigned to Neil, Sarah, or Tom. Every opportunity and project should always have a next task.

Who uses what

Neil: Add contacts, create opportunities, update the pipeline, send quotes, add tasks and notes after site visits. Primary sales and delivery.

Tom: Fully set up on laptop and phone. Uses the web form and Capsule app on site to add contacts, create opportunities, and attach photos. Field operations and quoting.

Sarah: View the pipeline to see what's coming in, check tasks assigned to her (mainly invoicing and customer care calls), track project status, assign tasks to Neil and Tom.

Everyone: Use the search bar at the top to find any contact, opportunity, or project instantly.

Capsule has a mobile app (search "Capsule CRM" in your app store). Use the same login details. It's useful for Neil and Tom when they're on site. Add notes, check tasks, look up contact details.
2

Adding contacts

▶

Contacts are the foundation. Every opportunity and project links back to a contact. Always add the contact first, then create the opportunity.

Adding a new contact

1
Click the + button (top right)
Choose "Add Person" or "Add Organisation".
2
Enter their details
Name, phone number, email address. If they belong to a company, start typing the organisation name. If it already exists, it will appear. If not, Capsule will create it.
3
Save
The contact is now in the system. You can add an opportunity to them straight away.
Always check if the contact already exists before creating a new one. Use the search bar at the top. Duplicate contacts cause confusion because you end up with notes and tasks split across two records.

What to record

At minimum: name, phone number, email. If it's a school or commercial client, add the organisation and the contact person's role. You can always add more details later. The important thing is to get the record created so you can link opportunities to it.

3

Sales pipeline

▶

The sales pipeline tracks every enquiry from first contact to close. The process is the same for all four services.

Pipeline stages

StageWhat it means
EnquiryAn enquiry has come in (email, phone call, website form, Facebook message)
Call / Site VisitYou've spoken to them or visited the site to assess the work
Proposal SentYou've sent the quote
Negotiation / Follow-upWaiting for a response, following up, negotiating terms
WonThey've said yes — the work is going ahead
LostThey've said no, gone quiet, or chosen someone else

Opportunity custom fields

FieldWhat to enter
Opportunity SourceWhere the enquiry came from: Renewal, Ad Hoc Existing Customer, Referral, Website, Facebook, Networking (Reel / Top Networking / Sterling), LinkedIn
Service InterestLandscaping, Fencing, Tree Work, or Grounds Maintenance
Enquiry DateWhen the enquiry first came in
Quote Sent DateWhen you sent the quote
You can view the pipeline as a list (like a spreadsheet) or as a board (cards in columns). Use whichever you prefer. The list view is better for scanning lots of opportunities. The board view is better for seeing the overall picture at a glance.

Filtering by owner

By default, you only see your own opportunities. To see everyone's, change the Owner dropdown to "Anyone". Sarah, this is how you see Neil's and Tom's opportunities.

Saved lists

You can save a filtered view as a named list for quick access later. Go to the list view of opportunities or contacts, set your filters, then click "Save List" at the top. Tick "Make list available to all users" so the whole team can see it. Current saved lists include "Recently Added Opportunities" (open opportunities sorted by newest first).

4

Creating an opportunity

▶

Every time an enquiry comes in, whether by email, phone call, website form, or Facebook message, create an opportunity.

1
Add the contact first
If they're not already in Capsule, add them as a person (and link to their organisation if applicable).
2
Create the opportunity
Click "Add Opportunity" from the contact record, or from the Sales Pipeline page.
3
Fill in the details
Name it clearly (e.g. "Landscaping - Sarah Lawrence" or "GM Renewal - Yenton Primary"). Add the expected value, source, service interest, and enquiry date.
4
Add a task
What's the next step? Follow up? Visit site? Send quote? Always have a task so nothing gets forgotten.
Never create an opportunity without linking it to a contact. An unlinked opportunity has no context, so you can't see who it's for or what the history is.

Multiple opportunities for the same client

If a client has multiple sites or multiple jobs (e.g. Miller Homes with Rugby, Swadlincote, and Colville), create a separate opportunity for each. They all link to the same organisation, but each job is tracked independently.

5

Winning and losing opportunities

▶

When you win

1
Drag the opportunity to Won (or click Edit and change the milestone)
A prompt will appear asking if you want to create a linked project.
2
Create the linked project
Click "Add linked project". Set the board to Customer Work. Choose the correct stage (Grounds Maintenance, Landscaping, Fencing, or Tree Work).
3
Set the dates
Start date (when the work begins) and expected close date (when the contract ends or the job finishes).
4
Add a task
What's the first thing that needs to happen? Schedule the work? Order materials? Send Sarah the details for invoicing?

When you lose

Mark it as Lost and select a reason (Price too high, No response, Chose competitor, etc.). Lost opportunities disappear from the pipeline but stay on the contact record. If the client comes back later with the same job, reopen the opportunity. If they come back with a different job, create a new one.

Don't be afraid to mark things as lost. A clean pipeline with only live opportunities is far more useful than one cluttered with dead enquiries. You can always reopen them.

Keeping the pipeline healthy

Go through your pipeline regularly and mark stale opportunities as Lost. Be honest: a quote from six months ago that was never followed up is not an active opportunity. If you think they might come back, mark the opportunity as Lost but add a check-in task on the contact record for 3–6 months' time. A clean pipeline gives you an accurate view of what revenue is genuinely in play.

Once an opportunity is won, the work is tracked as a project on the Customer Work board.

Creating a linked project

When an opportunity is Won, create a linked project: click the arrow next to Edit on the opportunity and select "Add Linked Project". Choose the Customer Work board and the correct stage. The project inherits the contact, organisation, and any notes from the opportunity.

Project workflow track

The Project Workflow track creates a checklist for each piece of work:

1
Send Deposit Invoice (Sarah)
2
Check Deposit Paid (Sarah)
3
Order Materials (Project Owner)
4
Confirm Start Date with Client
5
Start Work
6
Send Final Invoice (Sarah)
7
Quality Control Follow-up

Tasks unlock one at a time as you complete each step. If a step does not apply (no deposit needed), delete that task from the track on that project. The template is not affected.

Renewals

7

Tasks

▶

Tasks are the engine of the system. Every opportunity and every project should always have a next task. If there's no task, things get forgotten.

How tasks work

A task has a description, a due date, and an owner (Neil, Sarah, or Tom). When a task is due, the owner gets an email notification. When it's done, tick it off.

Common tasks

TaskWhoWhen
Follow up on quoteNeil / Tom3-5 days after sending
Check in with client (long wait)NeilEvery 6 months for dormant opportunities
Send invoiceSarahStart of month (GM) or on completion (projects)
Check client is happyNeil / TomMidway through contract
Customer care callNeil / SarahEvery 2 months (repeat task)
Renewal reminderNeilBefore contract end date (e.g. end of January for April-March contracts)

Repeat tasks

For recurring actions (like monthly invoice reminders or 6-monthly check-ins), set the task to repeat. When you tick it off, it automatically creates a new one for the next interval.

If you think of something that needs doing, even if it's weeks away, add it as a task immediately. Don't rely on remembering. That's the whole point of the system.

Task categories

Task categories help you filter your daily list. Three categories are set up: Urgent (red), Important (amber), Standard (green). When creating a task, set the category. Then filter by category in Calendar and Tasks to focus on what needs attention first.

Daily routine

Start each morning by opening Calendar and Tasks → Schedule view. This shows all tasks for today in an expanded view. If a task from yesterday was not completed, drag and drop it to today or to when you will actually do it. On the mobile app, the calendar is the third icon from the left. Tap a date to see that day's tasks.

8

Email sidebar & QuickBooks

▶

Email sidebar

Capsule can link to your email so you can store conversations against contact records. When you receive or send an email about a client, you can save it to their Capsule record with one click. This means Sarah can see what's been discussed without Neil having to forward emails.

QuickBooks integration

Capsule is already linked to QuickBooks. On any organisation record, you can see invoicing history: what's been paid, what's overdue, the total invoiced. This information syncs automatically based on the email address on the contact matching the one in QuickBooks.

You still do all invoicing in QuickBooks. Capsule just shows you the information for reference. Clicking on an invoice link takes you to QuickBooks to view the detail.

9

Transpond (email marketing)

▶

Transpond is the email marketing tool connected to Capsule. It's been set up and the domain verification has been sent to your web hosting provider. Once verified, you'll be able to send marketing emails from your 4sgs.co.uk domain.

What Transpond will do

Send targeted emails to groups of contacts. For example, upselling grounds maintenance to clients who've only used you for landscaping. You choose which contacts receive which emails by segmenting based on tags, service type, or any other data in Capsule.

Up to 1,000 active contacts are included in the free plan. Active contacts means the ones you're actually emailing, not everyone in Capsule.

Transpond is not set up for campaigns yet. It will be configured once the CRM data is clean and you're using Capsule consistently. For now, focus on getting the pipeline and projects working.

Capsule CRM notes

Won opportunity to project workflow: When you win a piece of work, the process is: mark the opportunity as Won, click the arrow next to Edit and select "Add Linked Project", choose the Customer Work board and the correct stage (Landscaping, Grounds Maintenance, etc.), save, then click "Add Track" and select the Project Workflow. This creates the full checklist of tasks from deposit invoice through to quality control.

Project workflow track: The track creates a sequence of tasks that unlock one at a time. Send the deposit invoice first. Once ticked, "Check Deposit Paid" appears. Then order materials, confirm start date, start work, send final invoice, and quality control follow-up. If a step does not apply (e.g. no deposit on a grounds maintenance contract), click the bin icon on that task to skip it. The remaining steps still work in sequence.

Tasks within a track can be customised for each project. Change the date, reassign to a different person, or add more detail. Changes only affect that specific project, not the track template.

Task categories (Urgent, Important, Standard) let you filter your task list by priority. Go to Calendar and Tasks, click the Category dropdown, and select the level you want to see. This cuts through the noise when you have a lot of tasks across multiple projects. You can also think of categories as business areas (Sales, Customer Delivery, Internal) instead of priority levels, but pick one system and stick with it.
